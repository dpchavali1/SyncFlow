## Recommendations

### For Quick Results (Paid)
1. **Screenshots**: Use mockuphone.com (free) + your actual screenshots
2. **Video**: Screen Studio ($89) - produces professional results with minimal effort

### For Free Option
1. scrcpy (Android) + QuickTime (Mac) + OBS for recording
2. Add device frames with mockuphone.com

## Recording Tips

```bash
# Install scrcpy (via Homebrew)

# Mirror and record Android screen
```
